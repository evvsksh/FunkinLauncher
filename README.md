**Funkin' Launcher** is a modern mod manager for *Friday Night Funkin’* that lets you browse, download, and install multiple mods seamlessly — powered by the GameBanana API.

---

## 🚀 Features

- 📦 Download and install FNF mods with one click  
- 🌐 Powered by the GameBanana API  
- ⚡ Fast, chunked downloader with pause/resume support  
- 🗂 Automatic mod extraction and folder management  
- 🧩 Supports multiple mods installed side-by-side  
- 🔄 Easy mod launching and switching  
- 💾 Local caching for faster reloads
